### 💰 Hệ thống tiền điện tử MyKoin

- Ví điện tử cơ bản
- Xây dựng trên Next.Js

### ⚙ Cài đặt

1. Cài đặt thư viện

```sh
  npm install
```

2. Chạy project

```sh
  npm run dev
```
